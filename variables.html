VARIABLES MAS INFLUYENTES
Se analizó la relación entre las variables climáticas y la variable objetivo RainTomorrow (¿lloverá mañana?). Los gráficos de tendencias revelaron patrones críticos para la predicción.
HUMEDAD A LAS 3PM
El predictor más fuerte identificado. Existe un umbral crítico cerca del 70% de humedad a partir del cual la probabilidad de lluvia al día siguiente aumenta drásticamente.
PRECIPITACION ACTUAL
La cantidad de lluvia del día actual (Rainfall) tiene una correlación directa con la probabilidad de lluvia al día siguiente (RainTomorrow).
PREPROCESAMIENTO Y PIPELINES
Antes de entrenar los modelos, se realizó un preprocesamiento riguroso para garantizar la calidad de las predicciones.
CORRECCION DE DATA LEAKAGE
Se eliminaron RISK_MM (precipitación del día siguiente) y Rainfall (precipitación actual) de las variables predictoras para evitar que el modelo "hiciera trampa" usando información del futuro.
STANDARDSCALER
Las variables numéricas se estandarizaron para que tengan media 0 y desviación estándar 1, evitando que variables con escalas grandes dominen el modelo.
ONEHOTENCODER
Las variables categóricas (Location, WindDir, etc.) se transformaron a formato numérico mediante codificación one-hot.
TRAIN/TEST SPLIT
Los datos se dividieron en 80% entrenamiento y 20% prueba, asegurando que la evaluación sea objetiva y representativa.
