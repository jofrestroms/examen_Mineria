METODOLOGIA DE TRABAJO
Para el desarrollo de este proyecto, se adoptó el marco de trabajo CRISP-DM (Cross-Industry Standard Process for Data Mining), garantizando que los resultados técnicos respondan verdaderamente a las necesidades del negocio.

Su implementación técnica se llevó a cabo utilizando el ecosistema de Python en Google Colab, apoyándonos en librerías como pandas para la manipulación del dataset, scikit-learn para el modelado y preprocesamiento, y matplotlib/seaborn para la visualización.
1. ENTENDIMIENTO DEL NEGOCIO
Comprender el problema: clima extremo en Australia impacta minería y logística. Necesidad de predicción de lluvia con 24h de antelación.
2. ENTENDIMIENTO DE DATOS
Dataset weatherAUS.csv con ~145K registros. Variables climáticas como temperatura, humedad, presión, velocidad del viento y precipitación.
3. PREPARACION DE DATOS
Limpieza de valores nulos, separación de columnas numéricas y categóricas, corrección de data leakage.
4. MODELADO + EVALUACION
Implementación de Árboles, Random Forest, Regresión Logística, Naive Bayes, SVM, Regresión Lineal y K-Means.
PREPARACION DE DATOS
El dataset weatherAUS.csv contiene registros meteorológicos de diversas ciudades australianas. Se realizó una limpieza de valores nulos y estandarización de variables.

Columnas numéricas: MinTemp, MaxTemp, Rainfall, Evaporation, Sunshine, WindGustSpeed, WindSpeed9am, WindSpeed3pm, Humidity9am, Humidity3pm, Pressure9am, Pressure3pm, Cloud9am, Cloud3pm, Temp9am, Temp3pm, RISK_MM

Columnas categóricas: Location, WindGustDir, WindDir9am, WindDir3pm, RainToday, RainTomorrow
